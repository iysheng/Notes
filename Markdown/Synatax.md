### MarkDown 笔记
---

1. 插入图片
``` markdown
![描述](图片路径)
![Fedora](https://fedoraproject.org/w/uploads/2/2d/Logo_fedoralogo.png)
```
2. 无序列表，使用 * 、- 、+ 作为列表标记符
``` markdown
* 行一
* 行二
- 行三
- 行四
+ 行五
+ 行六
```
3. 有序列表
``` markdown
1. 行一
2. 行二
```
4. 给文本添加超链接
``` markdown
[文本](web site 网址)
[fedora org](https://getfedora.org/)
```
5. 文本删除线和下划线
``` markdown
~~删除内容~~
<u>下划线</u>
```
